---

## üß© Features

- **Dynamic Shared Partials**  
  Update header, navigation, and footer from a single source.  
  All pages update automatically via lightweight client-side includes.

- **Modern Vite Workflow**  
  Fast dev server, optimized builds, ES modules, and clean structure.

- **Animation-Ready**  
  GSAP included for motion-rich hero sections, scroll interactions, and attention-grabbing UI.

- **Built for Future Growth**  
  Expand into services, portfolio, blog, or client portal without redesigning the entire foundation.

- **No Template Engine Overhead**  
  Pure HTML/CSS/JS for clarity and maintainability.
